# Ứng Dụng Quản Lý Công Việc Từ Xa

## Giới Thiệu
Ứng dụng quản lý công việc từ xa là một giải pháp toàn diện cho các freelancers và công ty nhỏ, giúp nâng cao hiệu suất và tối ưu quy trình làm việc. Với bảng điều khiển dễ sử dụng và các tính năng độc đáo, bạn có thể theo dõi tiến độ công việc, giao tiếp hiệu quả với các thành viên trong nhóm và lập kế hoạch dự án một cách tiện lợi.

## Tính Năng Nổi Bật
- **Theo Dõi Công Việc**: Giúp bạn quản lý và theo dõi tiến độ từng công việc.
- **Giao Tiếp Nhóm**: Tích hợp chat nhóm và bình luận trên từng công việc.
- **Lập Kế Hoạch Dự Án**: Cung cấp công cụ lập kế hoạch với lịch trình và deadline rõ ràng.
- **Thống Kê Hiệu Suất**: Cung cấp thống kê về hiệu suất làm việc của từng thành viên trong nhóm.

## Công Nghệ Sử Dụng
- React.js
- Node.js
- MongoDB
- Socket.io

## Cài Đặt
1. Clone repo này về máy của bạn:
   ```bash
   git clone https://github.com/user/app-remote-work-management.git
   ```
2. Cài đặt các gói cần thiết:
   ```bash
   npm install
   ```
3. Khởi động server:
   ```bash
   npm start
   ```
